Run a live prediction

This dashboard is a static HTML file and cannot execute the trained Random Forest, XGBoost and LightGBM models directly in the browser. To get a real prediction from the actual trained models, run the companion Streamlit application from a terminal in this project folder:

streamlit run app.py

The Streamlit app loads models/random_forest.pkl, models/xgboost_model.pkl and models/lightgbm_model.pkl directly and returns the genuine ensemble prediction for any district, date and price input — the same models evaluated in the Model Results page.
